Setp-gulp

Este repositorio contiene una configuración básica de Gulp y Sass para el desarrollo de temas personalizados en Shopify. La configuración proporcionada te permite compilar fácilmente archivos Sass en CSS, optimizar el código y ver los cambios en tiempo real gracias a BrowserSync.

Características
  • Compilación automática de archivos Sass a CSS.
  • Autoprefijado de CSS para garantizar la compatibilidad con varios navegadores.
  • Minificación de archivos CSS para una carga más rápida de la página.
  • Observación de cambios en los archivos Sass y recarga automática del navegador con BrowserSync.
  • Configuración lista para integrarse con temas de Shopify.
